# חמש אותיות

**חמש אותיות** הוא משחק ניחוש מילים בעברית, בהשראת המשחק Wordle.

## מטרת המשחק
לנחש מילה עברית בת **חמש אותיות** בתוך מספר מוגבל של ניסיונות.

## איך משחקים
לאחר כל ניחוש, האותיות מסומנות בצבעים:
- 🟩 האות נכונה ובמקום הנכון  
- 🟨 האות קיימת במילה אך במקום אחר  
- ⬜ האות אינה מופיעה במילה  

בעזרת הרמזים ניתן להתקדם ולגלות את המילה הנסתרת.

## התאמה לעברית 
המשחק מותאם במלואו לשפה העברית:
- כתיבה מימין לשמאל (RTL)
- מאגר מילים עברי
- חוויית משחק זהה ל־Wordle המקורי
